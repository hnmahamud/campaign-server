## Installation

For install all dependency:

```sh
npm install
```

## Configure .env file

For MongoDB

```sh
DB_USER=..........
DB_PASS=..........
```

For email send using nodemailer and gmail

```sh
EMAIL=..........
PASS=..........
```

## For run the project:

```sh
nodemon index.js
```

or

```sh
node index.js
```
