## Installation

For install all dependency:

```sh
npm install
```

## Create amd configure .env file

For MongoDB need database username and password

```sh
DB_USER=..........
DB_PASS=..........
```

For send email using nodemailer and gmail need email and app password

```sh
EMAIL=..........
PASS=..........
```

## For run the project:

```sh
nodemon index.js
```

or

```sh
node index.js
```
